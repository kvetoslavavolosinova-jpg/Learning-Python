WHAT IS A VARIABLE

A variable is a name used to store a value in Python.

You can imagine it like a box where you store information.

Each variable has a name so you know what is inside.

Variables can change, which is why they are called "variables".

---

TYPES OF VARIABLES

1. Numbers (int or float)
age = 39  → integer (whole number)
height = 1.59 → float (decimal number)

2. Text (string)
name = "Eva"
surname = "Novak"

3. Boolean (True / False)
is_student = True
has_dog = False

---

RULES FOR NAMING VARIABLES

- Can contain letters, numbers, and underscore (_)
- Cannot start with a number (wrong: 2age)
- Python is case-sensitive (age ≠ Age)
- Should be meaningful and describe the value stored
