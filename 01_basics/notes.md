Python	je programovaci jazyk, ktory pouzivame na pisanie presnych pokynov pre pocitac. 
	Je vysoko citatelny, co znamena, ze kod je lahko pochopitelny aj pre zaciatocnikov. 
	Ma jednoduchu a cistu syntax nema zlozite zatvorky, bodkociarkky atd. 

SYNTAX = GRAMATIKA
» súbor pravidiel, ktory urcije ako musi byt kod napisany - gramatika jazyka.
» V každom jazyku mame gramaticke pravidla tak isto ho ma ja Pathon aby pocita vedel co ma spravit. Pretoze pocitac nemysli, urobiiba to o co ho poziadame. 
» Ak porušíme pravidla v SYNTAX napíšeme zly kód = Python vypíše chybu. 

SPRAVNA SYNTAX: print("Hello, World!")
		- funkcia print sa vola so ()
		- text je v uvodzovkach ""
		- program sa spusti a vypise text

NESPRÁVNA SYNTAX: print"Hello, World!"
		- Python vypise SYNTAX ERROR 

Čistá syntax pythonu / cista gramatika Python-u zanmena, že nemusíš požívať bodkočiarky na konci riadku, žiadne zbytočné zátvorky
Toto všetko-tento syntax-túto  gramatiku vytvoril Guido van Rossum v roku 1991. Jeho cieľom bolo vytvoriť jednoduchý čitateľný jazyk, kt. bude vhodný pre začiatočníkov aj profesionálov.

Python je univerzálny jazyk a používa sa v mnohých oblastiach:

1. Webové aplikácie
   - Frameworky: Flask, Django
   - Použitie: tvorba web stránok a aplikácií

2. Analýza dát (Data Science)
   - Knižnice: Pandas, NumPy, Matplotlib, Seaborn
   - Použitie: spracovanie dát, vizualizácie, predikcie

3. Automatizácia a skripty
   - Opakujúce sa úlohy: presun súborov, spracovanie Excel/CSV, odosielanie e-mailov

4. Tvorba hier
   - Knižnice: Pygame
   - Použitie: jednoduché hry, vzdelávacie hry

5. Umelá inteligencia a strojové učenie
   - Knižnice: TensorFlow, PyTorch, scikit-learn
   - Použitie: rozpoznávanie obrázkov, chatboty, predikcie dát

6. Web scraping
   - Knižnice: Requests, BeautifulSoup
   - Použitie: získavanie dát z web stránok (ceny, články, recenzie)

7. Desktopové aplikácie
   - Knižnice: Tkinter, PyQt
   - Použitie: klasické aplikácie pre počítač
